Pro Paket – Yapay Zekâ Destekli Web Platformu, Admin Panel ve Mobil Uygulama Geliştirme Teklifi

Bu proje kapsamında, mevcut yapınızdan bağımsız olarak; sıfırdan tasarlanmış modern bir kurumsal web sitesi, yapay zekâ entegrasyonuna sahip merkezi bir admin panel (CMS + AI), aynı içerik altyapısını kullanan bir web uygulaması (web app) ve iOS & Android için yayınlanacak mobil uygulamalar geliştirilecektir.

Amaç; metinlerinizi, görsellerinizi ve yapay zekâ ile zenginleştirilmiş kelime açıklamalarını tek bir yerden yönetip, hem web hem mobil ortamda kullanıcılarınıza tutarlı, profesyonel ve genişletilebilir bir deneyim sunmaktır.

1. Proje Özeti

  • Sıfırdan tasarlanmış, modern ve çok dilli destek verebilen bir kurumsal web sitesi
  • Tüm metin ve görselleri yöneten merkezi bir admin paneli (CMS + AI)
  • Aynı içerik altyapısını kullanan web uygulaması (web app)
  • iOS & Android için yayınlanacak, web app ile uyumlu mobil uygulamalar
  • Metinler üzerinde yapay zekâ destekli kelime analizi ve açıklama üretimi

2. Proje Kapsamı

2.1. Kurumsal Web Sitesi

  • Modern, hızlı ve SEO uyumlu web sitesi tasarımı ve geliştirilmesi
  • Çok dilli altyapıya uygun yapı (örn. Arapça / Türkçe / İngilizce)
  • Kurumsal sayfalar (Hakkımızda, İletişim, Hizmetler vb.)
  • İçerik tarafında admin panel ile tam entegre yapı
  • Mobil, tablet ve masaüstü için tam responsive tasarım

2.2. Web Uygulaması (Web App)

  • Kullanıcı giriş sistemi (kayıt ol, giriş yap, şifremi unuttum)
  • Kullanıcı profil alanı (okunan metin sayısı, istatistikler vb.)
  • Metinlerin kategori → başlık → içerik mantığıyla listelenmesi
  • Metin içinde kelime tıklanınca ilgili kelimenin açıklamasını gösteren yapı
  • Tüm davranışların admin panel ve veritabanı ile entegre çalışması

2.3. Mobil Uygulamalar (iOS & Android)

  • iOS ve Android için tek kod tabanıyla geliştirilen modern mobil uygulama
  • Web app ile aynı içerik ve kullanıcı deneyiminin mobilde sunulması
  • Kullanıcı girişi, profil, metin listeleri ve kelime tıklama özelliği
  • Okunan metin sayısının takip edilmesi ve profil ekranında gösterilmesi
  • App Store ve Google Play’e yükleme ve ilk yayın sürecinin yönetimi

2.4. Admin Panel (CMS + AI Entegrasyonu)

  • Tüm metinlerin, başlıkların, kategorilerin ve görsellerin yönetimi
  • Yeni metin eklerken yapay zekâ entegrasyonu ile:
    • Metnin taranması
    • Kelime kelime veya seçili kelimeler için anlam / açıklama / özellik bilgileri üretilmesi
    • Bu bilgilerin sistemde yapılandırılmış formatta saklanması
  • AI çıktılarının admin tarafından düzenlenebilmesi, onaylanabilmesi
  • Kullanıcı listesi, kullanıcı detayları, okuma istatistikleri
  • Temel raporlama ekranları (toplam kullanıcı, toplam okunan metin vb.)

3. Yapay Zekâ Entegrasyonu

Proje kapsamında kullanılacak yapay zekâ motoru ile, admin panelde girilen metinler AI servisine gönderilecek; geri dönen sonuçlardan kelime bazlı açıklamalar ve özellikler çıkarılarak veritabanında saklanacaktır.

  • Metnin yapay zekâya gönderilmesi ve anlamlandırılması
  • Kelime veya seçili kelime grupları için açıklama / anlam / özellik üretimi
  • Bu verilerin JSON veya benzeri formatta saklanması
  • Web app ve mobil uygulamalarda kelime tıklama özelliği ile son kullanıcıya sunulması

Kullanılacak model ve servis (örneğin Gemini, OpenAI veya benzeri) proje başında sizinle birlikte seçilecek olup, lisans ve kullanım maliyetleri şeffaf biçimde planlanacaktır.

4. Teknik Mimari ve Teknolojiler

Projeyi uzun vadede sürdürülebilir, ölçeklenebilir ve modern bir yapı üzerine kurmak için güncel ve kurumsal standartlara uygun teknolojiler tercih edilecektir:

  • Backend & API: TypeScript tabanlı, NestJS benzeri modern bir backend framework
  • Veritabanı: PostgreSQL veya benzeri ilişkisel veritabanı
  • Kimlik Doğrulama: JWT tabanlı auth, rol bazlı yetkilendirme
  • Web Site & Web App: TypeScript ile Next.js benzeri modern React tabanlı yapı, Tailwind CSS ile responsive UI
  • Admin Panel: Tek sayfa uygulaması mantığında (SPA), backend API ile güvenli iletişim
  • Mobil Uygulamalar: Tek kod tabanından iOS & Android çıkışı alabileceğimiz React Native benzeri modern teknoloji

Bu teknoloji seçimi, hem geliştirme sürecini hızlandırmakta hem de uzun vadede bakım ve yeni özellik ekleme süreçlerini kolaylaştırmaktadır.
